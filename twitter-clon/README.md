# Twitter-Clon

Objetivo....

## Clanar el proyecto

> git clone ....

## Scripts del proyecto

### start

> npm start

### build

> npm build

Una vez realizado el build, copiar la carpeta `dist` al servidor productivo.

## Vistas y Rutas

- / - Login
- /tweets - Feed de todos los tweets
- /tweets/:id - Ver un tweet con sus comentarios
- /user/:id - Ver profile de una persona.

## Dependencias

- react
- react-dom
